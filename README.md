# Fading

```js
  import {fadingIn, fadingOut} from './fading';
```

## fadingIn
Display the matched elements by fading them to opaque.
```js
  fadingIn(item); 
    //or
  fadingIn(item, 300);
```


## fadingOut
Hide the matched elements by fading them to transparent.
```js
  fadingOut(item); 
    //or
  fadingOut(item, 300);
```
